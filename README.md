# Online Examination System (ASP.NET Core Web API)

## 🚀 Features
- JWT Authentication
- Role-based Authorization
- Exam Management
- Question Management
- Result Evaluation

## 🛠 Tech Stack
- ASP.NET Core (.NET 8)
- Entity Framework Core
- SQL Server

## 🧠 Architecture
Controller → Service → Data Layer

## 🔐 Security
JWT Authentication

## 📌 API Endpoints
- /api/auth/login
- /api/exam/create
- /api/question/add
- /api/result/submit
